# 🎉 项目介绍

Babyboom 是一个宝宝成长回忆记录平台，旨在帮助妈妈记录宝宝的成长点滴。该平台提供了一个简单易用的界面，允许妈妈创建和查看宝宝的成长记录。此外，您还可以 BOOM 一下，拆一份专门为您定制的记忆盲盒，让 AI 带您一起回顾美好的时刻。

# 🚀 技术栈

Java + SpringBoot + Mybatis-Plus + Mysql + RabbitMQ

调用 Qwen 视觉模型分析上传的图片，得到分析文本存入数据库。
